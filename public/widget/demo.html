Welcome to the demo page for the labor.fun embeddable chat widget. This is placeholder marketing copy so the floating chat bubble has something to float over. In a real deployment this would be your product landing page, docs site, or support portal.

Our AI assistant answers questions instantly, 24/7, right here on the page — no signup, no third-party chat vendor, fully self-owned. Click the chat bubble in the bottom-right corner to start a conversation.

Everything the assistant knows comes from your own knowledge base. It runs in an isolated container with per-visitor memory and does not share data with any outside chat provider.

Embed snippet

Adding the widget to any page is a single script tag:

<script
  src="https://your-host.example.com/widget/labor-widget.js"
  data-site-key="YOUR_SITE_KEY"
  data-endpoint="https://your-host.example.com:3100"></script>
